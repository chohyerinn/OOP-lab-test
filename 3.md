# UML Lab Exam Prediction Pack

This practice pack is based on the PDF lecture/homework files in `Downloads\객지프` and the Java projects in the Eclipse workspace. The problems are written in the likely exam style: an English specification plus a UML class diagram.

Assume Java, Eclipse, default package, and console I/O unless a problem says otherwise.

## Highest-Priority Topics

1. Classes with private fields, constructors, accessors, mutators, and test classes.
2. Static fields and static methods.
3. Arrays of primitive values and arrays of objects.
4. Two-dimensional arrays with nested loops.
5. Inheritance, `super`, overriding, abstract classes, and interfaces.
6. Exception handling with `try`, `catch`, `throw`, and `throws`.
7. Text file input/output with `Scanner`, `File`, `PrintWriter`, `FileReader`, and `FileWriter`.

```mermaid

```

Requirements:

- The constructor initializes the room number and initial people count.
- `totalNumber` must include the initial people count of every created room.
- `addOneToRoom()` increases both `peopleInRoom` and `totalNumber`.
- `removeOneFromRoom()` decreases both values only if `peopleInRoom > 0`.
- Write a demo that creates at least three room objects and prints each room count and the total.

Sample behavior:

```text
Add one to room A and three to room B.
Room A holds 1
Room B holds 3
Total in all rooms is 4
Remove one from both rooms.
Room A holds 0
Room B holds 2
Total in all rooms is 2
```
